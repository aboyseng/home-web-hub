# home-web-hub

## 项目简介

home-web-hub 是一个用于归档和发布多个独立 HTML 页面的仓库。这里存放着各种自包含的网页资源，每个页面均为独立的静态文件，不依赖于特定后端服务或域名。

本仓库的主要目的是提供一种简单、直接的方式来整理和分享那些用于个人用途或实验性质的网页内容。这些页面可能涉及工具、展示、学习笔记或其他轻量级 Web 应用。

## 目录说明

仓库的根目录下包含多个 HTML 文件，每个文件代表一个独立的页面。为了保持结构清晰，建议按以下方式组织：

- **根目录**：存放通用的单页面文件，适合快速访问的小型页面。
- **pages/**：用于存放更复杂的页面集合，可以包含子文件夹和关联资源（如 CSS、JavaScript、图片等）。
- **assets/**：共享资源目录，存放多个页面通用的样式表、脚本库或媒体文件。

当前目录结构示例：

```
home-web-hub/
├── index.html
├── page-example-1.html
├── page-example-2.html
├── pages/
│   └── sub-page/
│       ├── index.html
│       └── style.css
└── assets/
    ├── common.css
    └── common.js
```

## 页面归档说明

每个 HTML 页面应尽量自包含，即所有样式和脚本都内嵌或通过相对路径引用仓库内的资源。这有助于确保页面在克隆后或离线环境下仍能正常展示。

对于较大或依赖外部库的页面，建议在页面头部注释中注明所需的外部资源（如 CDN 链接），并尽可能将外部资源下载到本地 `assets` 目录中，以增强可移植性。
